# Induction Motor Dynamics Simulator

Interactive web-based simulator for modeling and analyzing the dynamic behavior of double-cage induction motors.

This project estimates torque, current, slip and acceleration time using equivalent circuit models and numerical methods, allowing engineers and students to evaluate motor performance under different load and starting conditions.

Built with Python, Flask, NumPy and Plotly, the platform provides:
- Motor and load parameter input  
- Torque and current curve visualization  
- Dynamic resistance modeling for double-cage rotors  
- Acceleration time estimation  
- Data storage and interactive charts  

This tool offers an accessible and cost-effective alternative to traditional engineering software for motor analysis and education.
